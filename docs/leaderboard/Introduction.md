# Leaderboards

## Introduction

Global and local leaderboards to enhance player competition. Supports top players, weekly/monthly rankings, and leagues.

## Features

- **Score Submission**: Submit numeric or string scores to leaderboards
- **Flexible Fetching**: Get top players or relative rankings around the current user
- **Iteration Support**: Handle leaderboard pagination with iteration tokens
- **Reset Information**: Know when leaderboards will reset (if applicable)
- **Caching**: Automatic local caching for offline access
- **Cancellation**: Full async operation cancellation support

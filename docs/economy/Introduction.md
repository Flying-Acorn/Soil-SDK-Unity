# Economy

## Introduction

The Economy module provides a comprehensive system for managing virtual currencies and player inventories. It allows you to track, update, and synchronize player assets across sessions.

Key features include:

- **Virtual Currencies**: Manage multiple types of in-game currencies (e.g., Gold, Gems).
- **Inventory Management**: Track player-owned items and their quantities.
- **Local Caching**: Offline access to player economy data via `EconomyPlayerPrefs`.
- **Automatic Synchronization**: Fetches the latest economy state upon initialization.
